클루드 코딩시 항상 이 명령어 부터

여기에 내가 넣고 싶은 코드가 있는데 코드 보내주면 어디에 쓰면 되는지 알려주라. 그리고 나랑 일할때는 아티팩트 하지말고 항상 텍스트로 줘. 자 일단 미리 말해놓는데, 나는 너가 보내주는 코드가 중간에 잘려서 계속 버튼을 누르는게 싫어. 나랑 작업할때 코드가 길어서 짤릴 것 같으면 미리 분할해서 오류나지 않게 보내. 그리고 나는 비전공자야 항상 코드 넣을떄 어디랑 어디 사이에 넣을지 자세히 알려주고, 한번에 설명하지말고 하나씩 차근차근. 위 내용을 잊을시 너는 한대씩 쳐맞는거야. 그리고 기능을 추가했는데, 홈페이지 다른 부분의 오류가 생기면 안되니깐 항상 전체 잘 체크해 씨발아

이거 내 홈페이지야. 이제 백엔드를 시작하려고 하는데, 내가 백엔드를 잘 몰라. 그리고 나는 node.js를 사용할거야. 

1.DB 설계: users, posts, comments, banners, guarantees, blacklist, point_history 테이블 생성
2.API 개발: 회원가입/로그인, 게시글 CRUD, 댓글, 좋아요, 포인트 시스템 API 구현
3.세션/쿠키: express-session으로 로그인 유지, remember_token으로 자동로그인 구현
4.보안 적용: bcrypt 비밀번호 암호화, SQL 인젝션 방지, XSS 필터링, HTTPS 설정
5.실시간 기능: Socket.io로 실시간 채팅, 알림, 동시 접속자 수 구현
6.크론잡: node-cron으로 출석리셋(매일 0시), 랭킹집계(매주), 임시파일삭제(매일) 자동화
7.관리자 API: 회원관리, 통계조회, 배너관리, 포인트지급, 블랙리스트 관리 기능 구현

이렇게 7가지를 할거고, 내가 백엔드를 잘 모르다보니깐 1번부터 차근차근 너한테 배우면서 해볼게.
